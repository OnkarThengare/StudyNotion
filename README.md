# StudyNotion

StudyNotion is a Full Stack EdTech platform where users can create, consume, and rate educational content.

## Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

## Features

- User Authentication
- Course Creation
- Course Enrollment
- Dashboard for instructors and students

## Installation

1. Clone the repository
   git clone https://github.com/OnkarThengare/StudyNotion.git

2. Install dependencies
   npm install

3. Run the project
   npm start
